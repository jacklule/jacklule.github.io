About this site

I am professor and chair in the Department of Journalism and Communication at Lehigh University in Bethlehem, PA, USA.

Explorations in GitHub is an attempt to evaluate GitHub as a content management tool for teaching, writing, publishing, sharing and archiving academic work in data visualization.

I had little experience in coding and none in programming. I figure that my learning curve will be similar to that of my students. I will attempt to leave a kind of "meta-commentary" that records my actions and decisions for the benefit of myself and others, beginning with the README page for this site.

Data Visualization Module

My primary motivation: In Fall 2016, I will be offering a seven-week module in Data Visualization as part of a larger class in Visual Communication. You can find here the Data Viz Syllabus site, which also has a substantive and detailed GitHub Set Up Guide.
